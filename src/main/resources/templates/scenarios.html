À propos du modèle de prédiction

Les estimations sont basées sur un modèle de régression linéaire entraîné sur les données 2014 du Grand Est. Le modèle prend en compte :

  • ✓ La température (facteur principal avec ~67% d'influence)
  • ✓ La saison (correction contextuelle)
  • ✓ Le type de jour (semaine vs week-end)
  • ✓ Les caractéristiques départementales
R² = 0.75 Coefficient de détermination
±120 MW Erreur moyenne

⚠️ Ces estimations sont indicatives et basées sur des données historiques. La consommation réelle peut varier selon d'autres facteurs non modélisés (activité économique, événements spéciaux, etc.).
